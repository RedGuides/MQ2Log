# /mqlogcustom

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/mqlogcustom <filename> <"message">
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Adds the "message" to Logs\Custom_File_Name_Here.Log (Mostly used by macro writers for debugging purposes)
<!--cmd-desc-end-->
